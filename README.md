# Punk Beers
Vue app that consumes a REST API ([punk API](https://punkapi.com/)).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
